# Cross-Device Communication Portfolio

This portfolio track captures the Kotlin-based local-network proof of concept
inside the wider Voice Hotkey repository.

## Included artifacts

- `CROSS_DEVICE_OVERVIEW.md`
- `CROSS_DEVICE_ARCHITECTURE.md`
- `CROSS_DEVICE_MOCKUPS.md`

## Proof of concept summary

- Kotlin Ktor server bound to `0.0.0.0:8080`
- Android presence detection over LAN
- Minimal browser UI for Android devices
- Signal receipt logged on the server
